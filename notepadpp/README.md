
## How to install
1. backup first old configs first: `%USERPROFILE%\AppData\Roaming\Notepad++`
2. copy new settings in.

## Functionality
- `Ctrl + 1` - Toggle document list
- `Ctrl + 2` - Toggle functions list
- `Ctrl + 3` - Toggle clipboard history
- `Ctrl + 4` - Toggle ASCII table
- `Ctrl + 5` - Toggle document map
